# Shadow Drift 🌑

A minimalist browser-based survival game. Keep your stability from collapsing by collecting glowing orbs and avoiding dangerous fragments — for 45 seconds!

## How to Play

- **W A S D** keys to move your white circle around the canvas
- Collect **cyan orbs** to gain +5 stability
- Avoid **red fragments** — they cost -10 stability
- Your stability slowly drains over time, so keep moving!
- Survive for **45 seconds** to win

## Win / Lose Conditions

| Condition | Result |
|---|---|
| Stability reaches 0 | 💀 "You Collapsed!" |
| Survive 45 seconds | ✅ "You Survived!" |

## Tech Used

- HTML, CSS, JavaScript
- Canvas API for rendering
- No external libraries needed
- Hosted on CodePen

## Running the Project on CodePen

Paste the entire file into CodePen's **HTML panel only** — no extra settings required.

> 💡 Click on the preview window first before using WASD keys, so the canvas captures your keyboard input.
