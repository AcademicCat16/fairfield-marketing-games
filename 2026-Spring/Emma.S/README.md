# 🖱️ Simple Clicker Game

> Click the button. Watch your score go up. That's it — and that's enough!

---

## 🎮 About

Simple Clicker Game is the purest form of a browser game — one button, one counter, infinite clicks. Every click adds 1 to your score. No timer, no goal, no game over. Just click.

---

## 🕹️ How to Play

1. Open the page
2. Click **Click Me!**
3. Your score goes up by 1
4. Keep clicking

---

## ⚙️ Game Details

| Stat | Value |
|------|-------|
| Starting score | 0 |
| Points per click | 1 |
| Win condition | None |
| Game over | Never |

---

## 🗂️ File Structure

```
simple-clicker/
├── index.html    # Body-only HTML — inline styles, score display, button
├── style.css     # Placeholder (styles inline in HTML)
├── script.js     # Click listener, score counter
└── README.md     # You are here
```

---

## 🖥️ How to Run

1. Place all files in the same folder
2. Open `index.html` in any modern browser
3. No installation or internet connection required
