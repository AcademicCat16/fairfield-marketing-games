## 🚨 IMPORTANT – Before You Paste the Code

Before copying the game code into CodePen:

1. Click the ⚙️ Settings button in CodePen
2. Go to the **JS** tab
3. Under **Add External Scripts/Pens**, paste:

https://unpkg.com/kaboom@0.5.0/dist/kaboom.js

4. Click **Save & Close**

If you do not complete this step, your game will not run.
