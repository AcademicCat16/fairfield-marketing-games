# ⭐ Pierinos cool StarPong

> "I made pong look cooler and made the game get faster and harder the longer you play."

A supercharged take on classic Pong — featuring a spinning star ball, rainbow trails, particle explosions, and glowing paddles that all speed up the longer you play!

## How to Play

### Main Menu
- Press **1** — 1 Player (vs AI)
- Press **2** — 2 Players
- Press **Q** — Quit

### Score Selection (2 Player only)
- Press **5** — First to 5 points
- Press **0** — First to 10 points

### Controls

| Player | Move Up | Move Down |
|---|---|---|
| Player 1 | **W** | **S** |
| Player 2 | **↑ Arrow** | **↓ Arrow** |

### Game Over Screen
- Press **R** — Restart
- Press **M** — Back to Menu
- Press **Q** — Quit

## Features

- Spinning star ball with rainbow glow trail
- Particle explosions on every hit and score
- Ball AND paddle speed ramp up over time — the longer you play, the harder it gets!
- Procedural sound effects (no audio files needed)
- 1 Player (vs AI) and 2 Player modes

## Tech Used

- HTML, CSS, JavaScript
- Canvas API
- Web Audio API for sound effects
- No external libraries needed
- Hosted on CodePen

## Running on CodePen

Paste into the three separate panels: HTML, CSS, and JS.

> ⚙️ **Important:** In the JS panel, click the gear icon and set **"Add Behavior"** to **"No wrap - bottom of body"** — otherwise the game will throw a null error on load.
