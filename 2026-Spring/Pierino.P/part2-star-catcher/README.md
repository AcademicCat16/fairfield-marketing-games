# ⭐ Starfall — Star Catcher

> "I made an arcadey game where you have to catch stars for a high score. I recommend using the mouse to move the platform most efficiently. The game gets harder the longer you last."

An arcade-style catching game — move your platform to catch falling stars before they hit the ground. The longer you survive, the faster the stars fall!

## How to Play

- Move your platform to catch the falling ⭐ stars
- Each caught star = **+1 score**
- Missing a star costs **1 life** — you have 3 lives total
- Survive as long as possible for a high score!

## Controls

| Action | Input |
|---|---|
| Move left | **← Arrow** or **A** |
| Move right | **→ Arrow** or **D** |
| Move (recommended) | **Click and drag** with the mouse |

## Features

- Stars fall faster the longer you play
- Glowing star and platform with motion blur effect
- Particle burst on every catch
- Score and lives HUD
- Restart button

## Tech Used

- HTML, CSS, JavaScript
- Canvas API
- No external libraries needed
- Hosted on CodePen

## Running on CodePen

Paste into the three separate panels: HTML, CSS, and JS.

> ⚙️ **Important:** In the JS panel, click the gear icon and set **"Add Behavior"** to **"No wrap - bottom of body"** — otherwise the game will throw a null error on load.
