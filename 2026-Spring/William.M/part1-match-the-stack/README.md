### JS Settings – External Script Required

This game uses **jQuery** to handle card clicks, shuffling, and animations.

To add it in CodePen:

1. Click the **gear ⚙️ icon** on the JS panel
2. Under **"Add External Scripts"**, paste this URL:

```
https://cdnjs.cloudflare.com/ajax/libs/jquery/3.7.1/jquery.min.js
```

3. Click **Save & Close**

> ⚠️ Without this, the game will show a black screen and throw a `$ is not defined` error in the console.
