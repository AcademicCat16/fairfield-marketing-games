# Apartment Designer ✨

A cute interactive drag-and-drop apartment decorating game built for **CodePen** using **HTML, CSS, and JavaScript**.

Players choose a character, read their short intro story, and help design their apartment by placing furniture into a blueprint-style layout. Once finished, the game shows a final decorated apartment with a celebration screen.

---

## Features

- Multi-screen game flow
  - Start screen
  - Character selection screen
  - Intro/story screen
  - Main design game screen
  - Final reveal screen

- Character-based experience
  - Each character has a unique name, style, and dialogue

- Apartment decorating gameplay
  - Add furniture items into the apartment
  - Drag and reposition furniture freely
  - Create your own layout

- Visual effects
  - Neon / dreamy UI style
  - Floating stars
  - Glitter celebration effect at the end

- CodePen-friendly structure
  - Separate HTML, CSS, and JS sections
  - No external framework required

---

## Technologies Used

- **HTML** for structure
- **CSS** for styling and animations
- **Vanilla JavaScript** for interactivity and game logic

---

## How to Use in CodePen

1. Open a new Pen in CodePen
2. Copy the project code into the matching sections:
   - HTML → HTML panel
   - CSS → CSS panel
   - JavaScript → JS panel
3. Run the project

No additional setup is required.

---

## Gameplay

1. Click **Play Now**
2. Choose a character
3. Read the character’s apartment design request
4. Click **Let’s Design!**
5. Add furniture from the left panel
6. Drag furniture around the apartment blueprint
7. Click **I'm Done Designing!**
8. View the final apartment reveal

## Project Structure

This project is divided into three parts for easy use in CodePen:

### HTML

### CSS

- page styling
- neon theme
- animations
- layout formatting
- furniture and final screen styles

### JavaScript
Contains:
- screen navigation
- character selection
- furniture generation
- drag-and-drop logic
- final reveal behavior
- glitter celebration effect
