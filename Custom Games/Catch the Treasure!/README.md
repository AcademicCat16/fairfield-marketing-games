# 💰 Catch the Treasure!

> Click the gold circle as many times as you can in 15 seconds!

---

## 🎮 About

Catch the Treasure is a simple browser-based clicking game. A gold circle jumps to a random position every time you click it — rack up as many clicks as possible before the timer runs out!

---

## 🕹️ How to Play

1. Open the page — the game starts immediately
2. Click the **gold circle** to score a point
3. The circle jumps to a new random position after each click
4. You have **15 seconds** — click as fast as you can!
5. When time runs out, your final score is shown in an alert

---

## ⚙️ Game Details

| Stat | Value |
|------|-------|
| Timer | 15 seconds |
| Circle size | 100 × 100px |
| Movement area | Up to 300px from top-left |
| Points per click | 1 |

1. Place all files in the same folder
2. Open `index.html` in any modern browser
3. No installation or internet connection required
