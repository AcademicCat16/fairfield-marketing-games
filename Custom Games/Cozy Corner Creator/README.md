# 🏡 Cozy Corner Creator

> Solve math problems, earn coins, and decorate your cozy room!

---

## 🎮 About

Cozy Corner Creator is a browser-based educational game that combines simple math practice with a fun room decorating mechanic. Answer math questions to earn coins, then spend them on furniture and decorations to fill your cozy corner!

---

## 🕹️ How to Play

1. Click **Start Chore** to get a math question
2. Type your answer in the input box and click **Submit Answer**
3. Correct answers earn **2 coins** 🪙
4. Spend coins in the shop to buy decorations for your room
5. Keep solving problems to unlock everything!

---

## 🧮 Math Chores

Questions are randomly generated using two numbers (1–10) and one of three operations:

| Operation | Symbol |
|-----------|--------|
| Addition | `+` |
| Subtraction | `-` |
| Multiplication | `×` |

Each correct answer earns **2 coins**. Wrong answers show a prompt to try again and generate a new question.

---

## 🛒 Decorations Shop

| Item | Cost |
|------|------|
| 🛋️ Sofa | 5 coins |
| 🌿 Plant | 3 coins |
| 💡 Lamp | 4 coins |
| 🖼️ Picture Frame | 6 coins |

Items appear in the room as soon as you purchase them. Each item is placed in a fixed position — no rearranging needed.

```
├── index.html    # Full HTML document with room and control panel

├── script.js     # Math generation, answer checking, coin system
└── README.md     # You are here
```

---

## 🖥️ How to Run

1. Place all files in the same folder
2. Open `index.html` in any modern browser
3. An **internet connection** is needed to load the decoration images from GitHub

---

## 🛠️ Tech Stack

- **HTML** — room layout, shop buttons, answer input
- **CSS** — cozy warm colour scheme, absolute-positioned room items
- **Vanilla JavaScript** — question generation, answer validation, coin tracking, item unlocking
- **External images** — decoration sprites hosted on GitHub

---

## 💡 Tips

- Multiplication questions can have large answers (up to 100) — take your time!
- You need **18 coins total** to buy all four decorations
- That means at least **9 correct answers** to fully decorate the room
- Items stay visible once purchased — there's no way to remove them
